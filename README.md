# Note Taker App

A website that allows you to create, organize, and view notes. 


## Instructions for Build and Use

Steps to build and/or run the software:

1. Navigate to this [page](https://landonedwards.github.io/note-taker-app/)

Instructions for using the software:

1. Write a note and click the save button.
2. Navigate to the View Notes page and view all saved notes there.
3. Click on any notes to edit them. They can be deleted by selecting the X button (there's no confirmation modal yet, so be sure you want to delete it before clicking it).

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* If you're using Visual Studio Code, make sure you have the node package manager installed. 
* Install TypeScript using npm
* Enter npm run preview or npm run dev to run the program
